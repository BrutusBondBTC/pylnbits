# pylnbits

python library of methods for accessing lnbits.com API or your own install.

Getting Started

```
git clone https://github.com/lightningames/pylnbits
cd pylnbits
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```
